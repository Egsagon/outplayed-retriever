# outplayed-retriever

Simple script to fetch favorite video pathes from Outplayed on any game.

License: MIT - See the `LICENSE` file.

### Usage

```sh
git clone https://github.com/Egsagon/outplayed-retriever
python3 outplayed-retriever/main.py
```
